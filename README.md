# Sandbox: Interactive code experiments

## 1 Particle letter "A" formation 
A particle cloud forms the letter "A" and respond to mouse movement. Repelled by the cursor, they gently drift back to their places, giving the letter an organic, 'living' feel.

**Tech Stack**: p5.js / JS / HTML / CSS
